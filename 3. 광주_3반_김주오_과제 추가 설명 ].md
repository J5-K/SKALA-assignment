 css를 이용해 소개 페이지로 돌아가는 버튼을 생성함


### 2부 Day1 ②여행지 소개 페이지

 구글 폰트 사용 및 style\_travel.css 구성해서 디자인

 전체 구조를 시맨틱 요소로 구성하고 <nav>를 position:fixed로 고정하고 링크를 각각 섹션 id로 연결

 클릭 시 부드럽게 해당 페이지로 이동하도록 설정, hover 시 색이 변하도록 설정

 첫 페이지 히어로 배경에 gradient 오버레이 + 이미지 설정

 "여행 시작" 버튼 flex로 보기 좋게 만들고 누르면 명소로 이동하도록 설정

 :root로 자주 사용하는 색상 설정해 페이지 디자인 구성

 Pseudo-elements(::after, ::before) 이용해 페이지 공백 제거 및 디자인

 명소 갤러리 figure+figcaption으로 구성하고 img마다 alt설정, display: grid 이용해서 정렬

 명소 사진을 따로 클래스로 저장해 페이지 크기에 따라 반응형으로 작동하도록 설계

 고흥 소개 페이지와 먹거리 페이지도 반응형으로 바뀌도록 설계

 명소 카드 hover 시 사진 크기 scale 커지도록 설정

 먹거리 섹션은 article과 팁 aside 2단 배치

 각각 추천 맛집 링크로 연결되도록 "평화국밥"과 "우주로돌짜장"에 링크 바로가기 생성함

 video 요소에 controls, poster 적용함

## 과제 2: 2부 ①반응형 상품 카드 갤러리

- 올 여름 보고 싶은 영화를 모아둔 페이지를 반응형 카드 갤러리로 만듦

- 반응형에서 3-2-1열이 되고 작아지면 메뉴가 숨겨지도록 설계

- 카드에 마우스 오버 시 위로 떠오르고 그림자가 진해짐

- 새로고침하면 아래에서 순차적으로 등장하도록 함

- flexbox이용해 상단바에서 로고가 왼쪽에 있고 메뉴와 버튼은 오른쪽으로 정렬함

- '영화 목록' 클릭하면 페이지 내 영화 목록으로 이동,

- '개봉 예정작'에 마우스를 가져가면 개봉예정작인 '스파이더맨'과 '오디세이' 2번 깜박임(DOM API 활용)

- '예매하러 가기' 클릭 시 예매할 수 있는 영화관 사이트로 이동

- DOM API 활용해서 라이트모드와 다크모드를 클릭해서 바꿀 수 있도록 설계

- :root 이용해 색상 var() 사용 및 display:grid와 flexbox를 이용해 보기 좋게 디자인

- 해당 영화 카드 오른쪽 아래 '영화 정보' 버튼 클릭시 해당 영화 정보 페이지로 이동
